About

IPTV Manager is an application for macOS and Windows for watching live television and recording programmes from legally obtained IPTV services. Built in Python and designed for ease of use, it brings a clean interface to IPTV playback without the clutter of third-party players.

Features

  • Live channel playback with full-screen support
  • Schedule and record programmes directly
  • Channel list management and favourites
  • EPG (electronic programme guide) support
  • Supports Xtream login details
  • Built in Python - lightweight and open to inspection
